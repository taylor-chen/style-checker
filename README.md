# style-checker

```bash
usage: style_guide.py [-h] [-e [EXTENSION]] inputDirectory

Style guide linter.

positional arguments:
  inputDirectory        Path to the input directory to lint.

optional arguments:
  -h, --help            show this help message and exit
  -e [EXTENSION], --extension [EXTENSION]
                        Change the extension of file to search for, the
                        default is `md`
```
